## 🤔 What is BijonAI?

Bijon(日语"ビジョン"的罗马注音)，译为"视觉"。BijonAI希望通过真正看得见的，可以交互的服务来让教育更公平，为学校，老师，学生的教学和学习提供帮助，让AI技术真正在教育上发挥巨大作用。
